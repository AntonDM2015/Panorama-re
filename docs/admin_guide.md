# Руководство администратора

Это руководство описывает, как управлять виртуальным туром по кампусу с помощью встроенной панели администратора.

## 1. Доступ к панели администратора
1. Перейдите по адресу `/admin` в вашем браузере.
2. Войдите в систему, используя учетную запись с ролью `admin` в базе данных.

## 2. Управление контентом

### 2.1 Добавление здания
Здания — это высший уровень организации (например, "Главный корпус", "Общежитие").
- Нажмите **"Добавить город"** (если нужный город еще не создан).
- Нажмите **"Добавить корпус"** под соответствующим городом.

### 2.2 Добавление локации
Локации представляют собой комнаты или конкретные области (например, "Аудитория 301", "Столовая").
- Нажмите **"Добавить локацию"** внутри раздела управления зданием.
- Вы можете указать этаж и тип (Локация/Комната).

## 3. Создание виртуальных туров

Основа режима "Street View" — настройка панорам и связей между ними.

### 3.1 Загрузка панорам
- Найдите локацию в панели администратора и нажмите **"Панорамы"**.
- Введите URL изображения панорамы (размещенного в Supabase или в другом месте) и название.
- Понятное название помогает идентифицировать панорамы при настройке переходов.

### 3.2 Настройка переходов (хотспотов)
Каждая панорама может иметь ссылки на другие панорамы.
1. В списке панорам нажмите **"Настроить переходы"**.
2. **Цель (Target)**: Выберите целевую панораму из выпадающего списка. Можно ссылаться на панорамы в других комнатах или зданиях.
3. **Направление (Direction)**: Введите угол в градусах (от 0 до 360), где должен появиться хотспот.
    - `0` — Вперед (Север).
    - `180` — Назад (Юг).
    - `90` — Справа (Восток).
    - `270` — Слева (Запад).
4. Нажмите **"Привязать"**.

### 3.3 Логика автоматической навигации
Если в одной локации несколько панорам, но вы **не настроили** связи вручную, система автоматически создаст:
- Хотспот "Вперед" (`heading: 0`) к следующей панораме в списке.
- Хотспот "Назад" (`heading: 180`) к предыдущей панораме.

Ручные связи всегда имеют приоритет над автоматическими для указанного направления.

## 4. Лучшие практики
- **Последовательное именование**: Называйте панорамы вроде "Коридор 1", "Коридор 2" и т. д.
- **Точные углы**: Чтобы навигация казалась естественной, совмещайте направление "Вперед" камеры с фактическим путем в коридоре или на улице.
